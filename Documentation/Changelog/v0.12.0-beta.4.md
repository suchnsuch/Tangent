+ Added support for `1)` and `A)` style ordered lists. These count as "large lists" in the same way as `*` unordered lists.
- Fixed an issue where trying to toggle on an inline style with a collapsed selection somewhat before an existing inline range of the same style could toggle _off_ that second style rather than adding a new inline range.
- Fixed an issue where clicking on the gutter of a note would select the bottom of the note rather than the beginning or end of the closest paragraph.
- Removed tooltips when hovering over embedded content to reduce obnoxious distraction.
